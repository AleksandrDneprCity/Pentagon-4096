BIOS ROM v4.2 для нового Pentagon-4096.
Выпуск 22.07.2026.

Переходная версия BIOS после версии 3.2 для Pentagon-1024 1.4, доработанного под Pentagon-4096.

History versions:

v3.2
Базовая версия BIOS для старой версии Pentagon-1024 1.4.

v4.2
- Адаптирован под новый Pentagon-4096, с учётом изменения раскладки портов BIOS, и с учётом увелниения объёма статического ОЗУ с 64 до 128 КБ.
- Тест статического ОЗУ расширен до 128 КБ и до 8 функций эмуляции ПЗУ Spectrum:
    Gluk Reset;
    TR-DOS;
    Menu-128;
    Basic-48;
    3+Menu-128;
    3+Sintax-128;
    3+DOS;
    3+Basic-48.

- Добавлена инициализация палитры стандартными цветами.
- Пока что фиксированное включение расширения основного ОЗУ до 4 МБ по стандарту Pentagon + Profi, с активацией возможности блокировки верхней памяти через порт #7FFD бит-5.
- Дополнил меню BIOS Setup / Статическое ОЗУ (пока что прописано старое название "Shady RAM") пунктами включения эмуляции новых страниц ПЗУ для режима Spectrum +3.
